    FANTOM1-2: cDNA Encyclopedia

    FANTOM1 — RIKEN full-length cDNA project launch

    FANTOM1 cDNA Mouse
    STAGE_PERIOD
    2000–2002
    DESCRIPTION
    FANTOM1 (Functional Annotation of the Mammalian Genome 1) launched by RIKEN. Created the first comprehensive full-length cDNA clone collection from the mouse genome, establishing the foundation for mammalian transcriptome annotation.
    URL
    https://fantom.gsc.riken.jp/

    FANTOM1-2 — Mouse full-length cDNA encyclopedia

    FANTOM2 cDNA Mouse Encyclopedia
    PUBMED_LINK
    12819125
    STAGE_PERIOD
    2003
    DESCRIPTION
    Construction of the mouse full-length cDNA encyclopedia, generating over 60,000 full-length cDNA clones. Represented the most comprehensive collection of mammalian full-length cDNAs at the time, enabling systematic annotation of the mouse transcriptome.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    Targeting a Complex Transcriptome: The Construction of the Mouse Full-Length cDNA Encyclopedia

    FANTOM3: Transcriptome Annotation

    FANTOM3 — Mouse gene catalog based on physical cDNAs

    FANTOM3 Transcriptome Mouse Annotation
    PUBMED_LINK
    16683036
    STAGE_PERIOD
    2005–2006
    DESCRIPTION
    FANTOM3 transcript annotation producing the mouse gene catalog based on physical cDNAs. Extended the cDNA collection and provided comprehensive transcript annotation, distinguishing protein-coding from non-coding RNAs and revealing the complexity of the mammalian transcriptome.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    Transcript Annotation in FANTOM3: Mouse Gene Catalog Based on Physical cDNAs

    FANTOM5+ & Web Resource

    FANTOM5 — Data collection & web resource

    FANTOM5 Data Collection Zenbu
    PUBMED_LINK
    28850107
    STAGE_PERIOD
    2017
    DESCRIPTION
    Systematic description of the FANTOM5 data collection, including CAGE profiling of 975 human and 152 mouse samples, transcription factor expression data, and the Zenbu genome browser interface. Data series underpinning the mammalian transcriptome atlases.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    The FANTOM5 collection, a data series underpinning mammalian transcriptome atlases in diverse cell types

    FANTOM Web Resource Database
    PUBMED_LINK
    30407557
    STAGE_PERIOD
    2018
    DESCRIPTION
    Major update of the FANTOM web resource, expanding to provide additional transcriptome atlases including FANTOM5 CAGE data, FANTOM6 lncRNA functional data, and improved tools for browsing and analyzing promoter/enhancer expression across cell types.

    https://fantom.gsc.riken.jp/
    TITLE
    Update of the FANTOM web resource: expansion to provide additional transcriptome atlases

    FANTOM — 20th year: transcriptomic atlases & ncRNA annotation

    FANTOM FANTOM6 ncRNA lncRNA
    PUBMED_LINK
    33211864
    STAGE_PERIOD
    2020
    DESCRIPTION
    FANTOM enters its 20th year with expanded transcriptomic atlases including FANTOM6 functional annotation of non-coding RNAs. The resource now covers CAGE-based expression for over 2,000 samples and provides extensive functional data on lncRNAs through systematic perturbation screens.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    FANTOM enters 20th year: expansion of transcriptomic atlases and functional annotation of non-coding RNAs

    FANTOM5: Promoter & Enhancer Atlas

    FANTOM5 — Promoter-level mammalian expression atlas

    FANTOM5 CAGE Promoter Expression Atlas
    PUBMED_LINK
    24670764
    STAGE_PERIOD
    2011–2014
    DESCRIPTION
    Flagship FANTOM5 paper: comprehensive promoter-level expression atlas across 975 human and 152 mouse samples using CAGE (Cap Analysis of Gene Expression) sequencing. Identified and quantified promoter usage across diverse cell types, revealing the core promoter architecture of the mammalian genome.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    A promoter-level mammalian expression atlas

    FANTOM — Paradigm shifts through the FANTOM projects

    Review FANTOM Genomics
    PUBMED_LINK
    26253466
    STAGE_PERIOD
    2015
    DESCRIPTION
    Review of the FANTOM projects and their paradigm-shifting contributions to genomics, including the transition from cDNA-based transcript annotation to CAGE-based promoter/expression analysis, and the discovery of widespread non-coding RNA transcription.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    Paradigm shifts in genomics through the FANTOM projects

    FANTOM5 — Atlas of human long non-coding RNAs

    FANTOM5 lncRNA CAGE Annotation
    PUBMED_LINK
    28241135
    STAGE_PERIOD
    2017
    DESCRIPTION
    Comprehensive atlas of 27,919 human long non-coding RNA (lncRNA) genes with accurate 5' ends defined by CAGE. Provided a high-confidence catalog of lncRNAs with detailed annotation of their expression patterns, conservation, and genomic features across diverse cell types.
    URL
    https://fantom.gsc.riken.jp/
    TITLE
    An atlas of human long non-coding RNAs with accurate 5' ends
